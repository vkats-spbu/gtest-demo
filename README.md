# Small example of the Google Test utilization

## Prerequisites

<par>sudo apt install build-essential</par>

<par>sudo apt install libgtest-dev</par>

## Build

### Demo application

<par> make </par>

### Demo tests suite

<par> make test </par>

## Run

### Demo application

<par> ./hello </par>

### Demo tests suite

<par> ./test-hello </par>
